# zuimeng -- 最盟

## 项目介绍

>本项目为最盟官微的后台，主要包括活动，社团，个人中心，在线报名等模块

## 技术架构

* 数据库 mongodb,mysql
* Web服务 node,express,nginx
* 构建 grunt
* 前端 zepto.js,sui,less,jade
* 版本控制 git

## 模块说明

* index, 首页模块
* mass, 社团
* activity, 活动
* self, 个人中心
* login/logout

## 开发说明

1. git clone
2. npm install
3. npm install -g grunt-cli
4. grunt
